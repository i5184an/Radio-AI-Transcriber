# Radio-AI-Transcriber
A custom-built AI-powered tool designed to automatically transcribe shortwave radio broadcasts into clear text. This project was realized with the support of Gemini AI, proving that powerful software can be developed even without formal programming expertise.

How it Works
This tool processes audio captured from radio transmissions and converts it into readable text logs, making it easier for DXers to document their catches.
Core Technology: It utilizes AI to transcribe spoken word content.
Language Models: It relies on external language models to achieve high accuracy in different languages.
Customizable: All files are provided as open-source Python scripts, allowing you to modify settings, paths, and configurations to suit your needs.

Prerequisites
Python 3.8 (or more) installed on your PC.

Libraries: You will need to install the necessary dependencies via your terminal:
pip install vosk (or the specific libraries used in your script).

Setup & Language Models
To enable the AI to recognize specific languages, you must download the appropriate language models.
Download Models: https://alphacephei.com/vosk/models
Configuration: After downloading, place the models in the designated folder within your project directory.

How to Launch
As this is a Python-based project, it is highly customizable. You can adapt the source code and the launch scripts to fit your own machine's environment.
Ensure your audio source file is ready.
Edit the Python script to point to your local directories and model paths.
Run the script using your terminal or a custom .bat file (similar to the logic used in other logbook projects).
